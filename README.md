# Deskripsi Tugas
Repository ini berisi implementasi algoritma fundamental oseanografi berdasarkan UNESCO Technical Papers in Marine Science No. 44 (1983). Program ditulis menggunakan bahasa Fortran 90 dan mencakup perhitungan:
1. Konversi Salinitas & Konduktivitas (SAL78)
2. Anomali Volume Spesifik (SVAN)
3. Kedalaman dari Tekanan (DEPTH)
4. Titik Beku (TF)
5. Panas Jenis (CPSW)
6. Adiabatic Lapse Rate (ATG)
7. Temperatur Potensial (THETA)
8. Kecepatan Suara (SVEL)
